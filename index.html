🕹️ About this software

Retro-Gamer brings classic gaming experiences to your ESP32 hardware. This application runs on your computer to assist with the setup and installation of emulators for your TTGO VGA or Olimex boards. You can play your favorite titles from systems like the Gameboy, NES, SNES, and Sega Genesis directly on a VGA monitor.

📋 What you need

Before you start, check that you have the following items ready:

  • A Windows computer running Windows 10 or newer.
  • A USB cable to connect your ESP32 board to the computer.
  • A TTGO VGA or Olimex ESP32 development board.
  • A working VGA monitor and a display cable.

📥 Download and Install

To begin, visit the releases page to download the latest version of the installer. This package contains everything required to communicate with your board.

Step 1: Visit this page to download: Retro-Gamer Release Page.

Step 2: Look for the file ending in .exe in the "Assets" section of the latest release.

Step 3: Click the file to save it to your computer.

Step 4: Find the file in your downloads folder and double-click it to start the installation process.

Step 5: Follow the on-screen instructions provided by the setup window to finish the installation.

🚀 Running the application

Once you install the software, you can launch it using the shortcut on your desktop or through the Windows Start menu.

  1. Connect your ESP32 board to your computer using the USB cable.
  2. Ensure your VGA cable connects your board to your monitor.
  3. Open the Retro-Gamer application.
  4. Select the correct communication port from the main menu.
  5. Choose the emulator you want to load from the list provided.
  6. Click the "Flash" button to transfer the software to your hardware.

The status bar at the bottom of the window will show the progress. Keep your device plugged in until the indicator says the process is complete.

🛠️ Troubleshooting

If you encounter issues, try these common fixes:

  • Device not found: Unplug and replug your USB cable. Try a different USB port on your computer.
  • Flash failed: Ensure no other programs are currently trying to talk to the ESP32 board.
  • No display: Check that your monitor is set to the correct input source. Verify the VGA connection.

If the software remains unresponsive, restart both the program and your ESP32 board by pressing the reset button on the board itself.

🧩 Supported Systems

Retro-Gamer currently supports several classic architectures through FabGL. This includes:

  • NES (Nintendo Entertainment System)
  • SNES (Super Nintendo)
  • Sega Genesis and Mega Drive
  • Sega Master System
  • Gameboy
  • Atari Lynx
  • PC Engine

Each emulator configuration is optimized for the ESP32 hardware to ensure steady frame rates and responsive controls.
